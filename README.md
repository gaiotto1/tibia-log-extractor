# Tibia Log extractor

Este projeto tem por objetivo extrair alguns dados de um log de uma partida de Tibia.
## Instalação

Clone o project:

```bash
  git clone https://github.com/gaiotto1/tibia-log-extractor.git
```

Instale as dependências:

```bash
  npm install
```

Rode o projeto:

```bash
  npm start
```
